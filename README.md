# 🧠 AI Foundations

### Learn Artificial Intelligence from First Principles
---

Welcome to **AI Foundations**!

This repository documents my journey of learning Artificial Intelligence from the ground up.

Instead of only building AI applications, my goal is to understand **why they work**, **how they work**, and **how to implement them from scratch**.

Every topic in this repository includes:

- 📖 Concept Explanation
- 🧠 Intuition & Real-World Analogies
- 🧮 Mathematical Foundations
- 🐍 Python Implementation
- 📓 Jupyter Notebook
- 📊 Visualizations
- 📝 Key Takeaways
- 📚 References

Whether you're a beginner or someone revisiting the fundamentals, I hope this repository helps you learn AI one concept at a time.
